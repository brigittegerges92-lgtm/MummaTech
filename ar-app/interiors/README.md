# Interior Images

Drop your 5 reference interior photos into this folder using the filenames below. The Studio loads them automatically on page open.

| Filename | Reference scene | Light direction (auto) |
|---|---|---|
| `interior-1.jpg` | Brick & sun loft — exposed brick, industrial windows, tan tufted sectional | Upper-right window, strong directional warm sun |
| `interior-2.jpg` | Federation green velvet — fiddle leaf, plaster ceiling, parquet | Right windows, soft warm |
| `interior-3.jpg` | Coastal golden hour — cream boucle sofa, ocean view, olive tree | Right balcony, warm golden side |
| `interior-4.jpg` | Federation marble — fireplace, stained glass arched doors | Right arched windows, directional warm |
| `interior-5.jpg` | Coastal harbour living — white panelling, rattan pendant, bay view | Right bay, diffuse cool |

Each room ships with a tuned **lighting recipe** that re-tints the artwork to match the room's actual light (warm directional in the brick loft, diffuse cool in the harbour view). To adjust, edit `LIGHT_RECIPE` in `index.html`.

If a file is missing, the slot stays blank and you can drop an image straight into it from the Studio (or the AR app's right-rail upload zones).
